# pascalize

> Turns a string into PascalCase.

# Install

```bash
$ npm i -S pascalize

# or with yarn

$ yarn add pascalize
```

## Usage

```js
import pascalize from 'pascalize';

pascalize('foo bar');
// -> FooBar

pascalize('john-smith');
// -> JohnSmith
```
